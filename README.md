# COMPAC-Project
Place holder repository to link to the implementation on the GitLab-Instance at https://gitlab-secsys.lit-systems.jku.at/ak124238/prj_lit-compac.

This project focuses on generating and reducing NP problem instances to PARTITION problem instances and solving those subseqently using analog circuits.
